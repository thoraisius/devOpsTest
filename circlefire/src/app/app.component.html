Welcome to DevOps!

Let's setup a continuous deployment pipeline!

Setting up

Create GitHub account and a repository for your project

Clone or download github.com/bjornnorgaard/orbit

Copy and paste the folder 'circlefire', into your own project

Deploy to Hosting

Sign into firebase.google.com

Create firebase project. The project id will be your unique URL

Go to the Develop > Hosting > Get Started

Follow setup dialog for hosting using powershell, choosing

  • Default Project: self-explanatory
  • Public directory: dist/circlefire
  • Configure as SPA: YES

    • Update package.json scripts with these

    "build": "ng build --prod",

    "deploy": "ng build && firebase deploy",

    Deploy the app using your new command

    npm run deploy

    Now the app should be available to the internet!

    Automate deployment

    Sign into circleci.com using github login

    Add your github project and follow guide for Linux/Node

    Copy config.yml content from here

    Install firebase tools as a devDependency

    npm i --save-dev firebase-tools

    Get a firebase access token for the circleci to use

    firebase login:ci

    Add this token to your circleci environment as FIREBASE_TOKEN


    Push new commit and behold!
